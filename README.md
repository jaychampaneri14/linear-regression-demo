# Linear Regression Comprehensive Demo

A complete tutorial covering five variants of linear regression with visualizations.

## Topics Covered
1. **Simple Linear Regression** — slope, intercept, R²
2. **Multiple Linear Regression** — multi-feature prediction
3. **Polynomial Regression** — degree comparison, overfitting demonstration
4. **Regularization** — Ridge, Lasso, ElasticNet coefficient paths
5. **Learning Curves** — bias-variance tradeoff visualization

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Output
- `simple_lr.png` — scatter + regression line
- `multiple_lr.png` — actual vs predicted
- `polynomial_lr.png` — degree comparison
- `regularization_paths.png` — Ridge and Lasso coefficient paths
- `learning_curves.png` — training size vs R² curves
